第一步：我們要提供防疫包給誰？

在推出任何產品前，企業都會先提出「市場的需求為何？」、「消費者在哪？」等問題，以了解、並滿足使用者。

同樣概念套用於政府，我們也須在推出服務前，先了解使用者的需求，以獲得更高的大眾滿意度及形象。

Insights
策略洞察
  • 防疫包內容物不一致導致民眾比較心態，引發負面情緒。
  • 防疫包內容物無法滿足需求，導致使用者認定為無效。
  • 呈現形式未考慮使用者需求，造成不便，產生負面情緒。
User’s needs
策略洞察 - 防疫包三層次需求
  • 初階
    • 品項：基本防疫物資（酒精、消毒水、口罩）
    • 目標：避免過度期物資短缺出現負面情緒
  • 中階
    • 品項：食物、零食 (泡麵、餅乾) 等
    • 目標：避免出現期望落差
  • 高階
    • 品項：心靈層面療癒小物、生產力工具
    • 目標：強化與防疫包的關聯與正面情緒

第三步：我們如何更有效傳遞防疫包概念？

合適的呈現形式能加分地讓內容更有效地傳遞，而不合適的則可能導致內容產生扣分效果。

好的呈現形式是重要的，而防疫包呈現最重要的是「通用」，意即「無須特別改良便能給所有人使用」

字體使用建議 Typography usage
在各種尺寸、場合上都萬用，簡單乾淨的黑體
不過度可愛，親和而正式的圓體
視覺上有重量依然好閱讀，粗細適中的粗體
勿用過多細節的字體，字體較小時不易閱讀
勿用裝飾性過強的字體
勿用手寫字體，易造成閱讀不便
色彩使用建議 Colour usage
選擇二至五個顏色，以明亮、溫和、接受度高為主。
勿選擇過於強烈的顏色。尤其是易帶來壓迫感的亮紅色與亮黃色！
避免明度過低、過於暗沉的顏色
避免單用易引發政治、宗教聯想的顏色
需符合視障者友善原則，盡量不要同時使用紅/綠/棕/橘
角色設計建議 Character design usage
用可愛角色溝通嚴肅的必要資訊，能降低溝通門檻（以口罩小子為例）
  • 先選擇：與防疫相關，但無負面資訊的物件
  • 再創作：輔助為主要目的。勿過度複雜、鮮豔，以簡單、幾何圖形為主
  • 後應用：傳遞訊息為目標，勿搶資訊焦點
包裝形式建議 Packaging usage
具別種功能的包裝，能有效提升防疫包使用度
不只是包裝： 除了裝防疫物資外，或許還能結合體溫登記...
對環境好：需大量發放，可使用環保可分解材質或可重複使用的包裝
傳達資訊：印上需要讓隔離者知道的重要資訊，減少傳單成本、同時增加「防疫包」的使用
文字表達 Copy writing
使用統一的標註方式
使用合適圖示/列點取代段落式文字說明
勿同時以多種方式標註重點
避免專業語彙的使用
